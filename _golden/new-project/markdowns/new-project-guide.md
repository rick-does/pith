# Getting Started

Welcome to your new PiTH project. A quick tour of what you can do.

## Add files

- **⋮ → File → New file** — create a fresh `.md` file
- **⋮ → File → Add Existing File** — pick existing `.md` files from anywhere on disk
- Files not yet placed in the hierarchy appear in the **Unlinked** pane; drag them into the tree to organize

## Organize

- Drag an item to reorder; drop it onto another item to nest
- **⋮ → View YAML** to edit `tree.yaml` directly
- **⋮ → Flatten hierarchy** / **Restore hierarchy** — undo a flatten

## Edit

- Click a file to open the editor with live preview
- YAML frontmatter at the top of a file shows as editable fields
- `:w` or `:x` to save in vi mode; `Ctrl+S` always works

## Export & analyze

- **⋮ → Export to…** — MkDocs or Docusaurus site config
- **⋮ → View HTML/PDF** — single-document export of the whole project
- **⋮ → Scan Project** — readability and prose stats for every file

When you don't need this page anymore, toggle it off from **⋮ → Settings → Hide new project file**.
